1. python 正则表达式的使用
    - 参考链接：[菜鸟教程](https://www.runoob.com/python/python-reg-expressions.html) [在线工具]()
    - 两种使用形式
        ```python
        #第一种
        re.match(pattern, string, flags=0)
        #第二种
        pattern = re.compile(r'\d+')                    # 用于匹配至少一个数字
        m = pattern.match('one12twothree34four')        # 查找头部，没有匹配
        ```
    - 多种方法
        ```python
        match() #从开头匹配
        search() #任意位置匹配  
        findall() #返回全部匹配的结果[str列表]，其他只返回一次
        sub() #替换匹配项
        () #()将不确定项例如 .*? 用()括起来可以在 返回结果 中单独找出 不确定项
        split() #以匹配项为分隔符，分割str
        ```
    - 返回值的类型
        ```
        1.str 或 str[]
        2.某种对象，可以使用group(),groups(num=)
        group()     #返回 str
        groups()    #return 一个元组或对应的匹配项，（需要在正则表达式中搭配()使用）
        ```
